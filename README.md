## Description

This project is a weather web application that displays real-time and forecast weather data, including:

- Temperature
- Humidity
- UV Index
- Sunrise and sunset times
- Hourly forecast
- Daily forecast

Weather data is provided by [OpenWeather](https://openweathermap.org/).

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
- Google Maps Places API
- Google Maps Time Zone API
- OpenAI API
